# くらげふわふわキャッチ

深海で上昇する半透明のくらげを、ふわふわの泡で包んで捕まえるカジュアルキャッチゲームです。

## 起動方法

```bash
npm start
```

ブラウザで次を開きます。

```text
http://127.0.0.1:5173
```

## 公開用ビルド

Codex Sites などの静的ホスティングへ渡す公開ディレクトリは `dist/` です。

```bash
npm run build
npm run preview
```

プレビューは次のURLで確認できます。

```text
http://127.0.0.1:4173
```

公開設定のメタデータは `codex-sites.json` にまとめています。

## プレイ方法

1. 画面をクリック、タップ、またはドラッグするとゲームが始まります。
2. 泡は画面内を上下左右・斜めに移動できます。ドラッグ、スワイプ、またはキーボードで動かします。
3. 上昇してくるくらげと泡を重ねると捕獲できます。
4. 捕獲するとスコアが増え、短時間で続けて捕まえるとコンボボーナスが入ります。
5. 7秒ごとに強い風向きが変わり、後半ほど速いくらげが混ざります。HUD右側の矢印とゲージで風向きと強さを確認し、少し早めに先回りしてください。
6. 39秒経過すると `Game End` です。
7. くらげを5匹逃すと `Game Over` です。

## 操作

- スワイプ / ドラッグ: 泡を上下左右・斜めに移動
- `↑` / `↓` / `←` / `→` または `W` / `A` / `S` / `D`: 泡を上下左右・斜めに移動
- `Space`: リスタート
- `M`: ミュート切り替え
- `F`: フルスクリーン切り替え

## 風の見方

ゲーム開始から最初の7秒間は無風です。その後は7秒ごとに風向きが切り替わり、くらげが風の方向へ大きく流されます。

HUD右側の `Wind` 表示で、現在の風向きと強さを確認できます。

- `Wind - CALM`: 無風です。くらげは風では左右に流されません。
- `Wind < LEFT`: 左向きの風です。くらげが左へ流されます。
- `Wind > RIGHT`: 右向きの風です。くらげが右へ流されます。
- 横ゲージ: 風の強さです。ゲージが長いほど、くらげの横流れが強くなります。

風が強いときは、くらげが泡に届く前に横へずれていきます。後半は出現間隔も短くなるため、矢印の向きを見て、泡を少し先回りさせると捕まえやすくなります。

## BGM

- `assets/BGM/deepsea-whale-2.mp3` をゲームBGMとして使用します。
- ブラウザの自動再生制限に対応するため、BGMはクリック、タップ、ドラッグなどの開始操作後に再生されます。

## コツ

- 泡は少し遅れてなめらかに追従するため、くらげの進路を先読みして動かすと捕まえやすくなります。
- 風が強く吹いている間は、全てのくらげが矢印方向へ大きく流されます。HUDの風向き矢印を見て、泡の位置を先回りさせてください。
- 連続捕獲を狙う場合は、中央付近で待ちつつ上下にも位置を合わせると複数レーンに対応しやすくなります。
